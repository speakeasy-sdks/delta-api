# TransactionStatus

One of success, failed, pending, canceled, refunded, or other.


## Values

| Name       | Value      |
| ---------- | ---------- |
| `Success`  | success    |
| `Failed`   | failed     |
| `Pending`  | pending    |
| `Canceled` | canceled   |
| `Refunded` | refunded   |
| `Other`    | other      |
