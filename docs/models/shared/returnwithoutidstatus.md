# ReturnWithoutIdStatus

Status of the order return. Can be one of announced, transit, delivered, canceled, completed, or other.


## Values

| Name        | Value       |
| ----------- | ----------- |
| `Announced` | ANNOUNCED   |
| `Transit`   | TRANSIT     |
| `Delivered` | DELIVERED   |
| `Canceled`  | CANCELED    |
| `Completed` | COMPLETED   |
| `Other`     | OTHER       |
